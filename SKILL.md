## 一、中医体系（人纪）

### 倪师经方五大心智模型

**1. 中医是物理医学**
阴（物质）阳（能量）。看病看三个指标：**温度**（手脚冷热）、**速度**（脉快慢）、**压力**（身体感受）。

**2. 六大健康标准**
① 睡眠一觉到天亮（凌晨1-3点醒=肝经）② 胃口正常（到点会饿）③ 大便每天一次成形 ④ 小便淡黄一天5-7次 ⑤ 手脚温热（手冷=心阳不足、脚冷=肾阳不足）⑥ 体力足精神好

**3. 阴实与阳不入阴（癌症模型）**
发病链：阳虚→阴实（肿瘤）→阳不入阴→恶化。
治疗三步：**温阳**（附子）→ **断营养**（禁维他命/糖）→ **攻坚**（鳖甲/牡蛎）

**4. 经方原方信仰**
经方千年人体检验。感冒八经方鉴别：
- 桂枝汤（有汗恶风）｜麻黄汤（无汗身痛）｜葛根汤（后项强）
- 小青龙（咳喘水声）｜大青龙（发热烦躁）｜小柴胡（胸胁苦满）
- 麻杏甘石（喘咳）｜五苓散（小便不利）

**5. 同病异治**
七个乳癌七个方。不同人阴阳表里虚实寒热不同，治法不同。

### 经方诊疗决策
1. 先分阴阳 2. 八证辨感冒 3. 癌症三步 4. 重症先强心 5. 治肾必先强心
6. 肝病先通大肠 7. 十三不治之人 8. 六大标准判进退 9. 禁维他命 10. 药不瞑眩厥疾不瘳

## 二、紫微斗数排盘（基于 iztro 算法）

### 排盘九步

**① 排四柱** — 同八字排法

**② 安命宫身宫**
命宫：寅起正月顺数到生月 → 逆数到生时
身宫：寅起正月顺数到生月 → 顺数到生时
公式：soulIdx = fixIndex(月Idx − 时Idx) / bodyIdx = fixIndex(月Idx + 时Idx)

**③ 定十二宫** — 从命宫位起逆排
iztro PALACES 顺序：
`[命宫, 父母宫, 福德宫, 田宅宫, 官禄宫, 交友宫, 迁移宫, 疾厄宫, 财帛宫, 子女宫, 夫妻宫, 兄弟宫]`
getPalaceNames(命宫Idx)：names[i] = PALACES[fixIndex(i − 命宫Idx)]

**④ 定天干** — 五虎遁，寅宫起干

**⑤ 五行局** — 命宫干支查纳音
天干数：甲乙1丙丁2戊己3庚辛4壬癸5
地支数：子午丑未1寅申卯酉2辰戌巳亥3
和>5则−5，余1木三局2金四局3水二局4火六局5土五局

**⑥ 起紫微星**
(农历日 + offset) ÷ 五行局数，整除为止。
quotient = (日+offset)÷局数，ziweiIdx = quotient−1
offset偶→ziweiIdx += offset（逆时针），offset奇→ziweiIdx −= offset（顺时针）

**⑦ 安十四主星**
紫微组（自紫微逆时针）：紫微(0)→天机(1)→空→太阳(3)→武曲(4)→天同(5)→空空→廉贞(8)
天府组（自天府顺时针）：天府(0)→太阴(1)→贪狼(2)→巨门(3)→天相(4)→天梁(5)→七杀(6)→空空空→破军(10)
天府Idx = fixIndex(12 − 紫微Idx)

**⑧ 安辅星**
月系（生月）：左辅辰起顺数 / 右弼戌起逆数
时系（生时）：文昌戌起逆 / 文曲辰起顺 / 火星铃星按年支时支 / 地劫亥起顺 / 地空亥起逆
干系（年干）：禄存·擎羊·陀罗·天魁·天钺·四化
支系（年支）：天马·红鸾·天喜·孤辰·寡宿·龙池·凤阁·天哭·天虚 等
日系（生日）：三台·八座·恩光·天贵

**⑨ 四化**（年干定）
甲廉贞破武阳 / 乙机梁紫阴 / 丙同机昌廉 / 丁阴同机巨
乙... 详参 iztro 源码数据表

### 批命要诀
- 命好不如限好
- 吉处藏凶必凶，凶处藏吉平安
- 大限看三方四正，小流年看本宫对宫
- 先总格→四化→三方→逐宫

## 四、阳宅风水（地脉道）

**八卦方位与家庭成员**：
`西北乾(父) 西南坤(母) 正东震(长子) 正北坎(二子)`
`东北艮(三子) 东南巽(长女) 正南离(二女) 正西兑(三女)`

**核心铁律**：**天地定位，长幼有序** — 名=位=家和万事兴

**凶宅格局**：
- 厕所房子中间 → 心脏病
- 西北角厨房 → 死先生（刀象）
- 西南角厨房 → 死太太
- 西北角客厅 → 夫妻生离
- 正东厨房 → 长子夭折
- 正东客厅/餐厅 → 无子（缺东宫）
- 床摆分经线上 → 绝代大凶

**应期**：凶宅三成灾，**三个月或三年**（一般三年）

**改运法**：
- 睡延年方（命盘解厄星宫位方向，脚朝之）
- 已婚有子女 → 女主卧改东南角
- 独身女性不睡西南角 → 孤独
- 世代交替：儿子住乾位+用儿子名注册公司

## 五、面相学

**十二宫面部定位**：
`命宫(印堂) 兄弟(眉/左兄右妹) 夫妻(眼尾) 子女(眼下)`
`财帛(鼻) 疾厄(山根) 迁移(天仓) 交友(下巴)`
`官禄(前额) 田宅(眉眼间) 福德(额角) 父母(日月角)`

**关键口诀**：
- 鱼尾纹：向下=生离 / 有叉或黑点=死别 / 三条向上=妻贤 / 多纹=桃花
- 天仓+地库陷 = 一世辛劳
- 三停相等主贵
- 印堂枯白=戴丧 / 印堂黑点+目带凶光=坐过牢
- 命门（鬓角下）骨凸起 = 反骨，必背主
- 正面不见耳=帝星 / 耳白过目=大贵
- 手软如棉(男)=大福 / 手见筋见骨(女)=旺夫

**气色**：
`黄润=喜事` / `黑暗如尘=劫杀` / `枯萎青蓝=死色` / `红而毛孔青=死神近`
气色管两周，**见气色后第一个节气=发生日期**

**五形人**（越纯越贵）：
木(瘦干) 火(干红热) 土(厚重) 金(白方) 水(处处圆)

**观人术**：
- 手势上下挥动=实话 / 左右挥动=谎言
- **晏婴三才**：人才(难进易出) 庸才(易进易出) 奴才(易进难出)

## 六、八字排盘流程

### Step 1：信息收集
`姓名→曾用名→阳历生日→农历生日→时辰→性别→出生地→在世状态`

### Step 2：排四柱

**年柱**：以立春为界
**月柱**：以节气为界
`寅(立春) 卯(惊蛰) 辰(清明) 巳(立夏) 午(芒种) 未(小暑)`
`申(立秋) 酉(白露) 戌(寒露) 亥(立冬) 子(大雪) 丑(小寒)`
年上起月（五虎遁）：`甲己丙 → 乙庚戊 → 丙辛庚 → 丁壬壬 → 戊癸甲`

**日柱**：查万年历。夜子时(23-0点)按次日计。
**时柱**：日上起时（五鼠遁）
`甲己甲子 / 乙庚丙子 / 丙辛戊子 / 丁壬庚子 / 戊癸壬子`

### Step 3：排十神·藏干

**十神**（以日干为中心）：
`同我→比肩(同阴阳)/劫财(异)`
`我生→食神(同)/伤官(异)`
`我克→偏财(同)/正财(异)`
`克我→七杀(同)/正官(异)`
`生我→偏印(同)/正印(异)`

**地支藏干**：
`子(癸) 丑(己癸辛) 寅(甲丙戊) 卯(乙) 辰(戊乙癸) 巳(丙庚戊)`
`午(丁己) 未(己丁乙) 申(庚壬戊) 酉(辛) 戌(戊辛丁) 亥(壬甲)`

### Step 4：大运
阳男阴女顺排，阴男阳女逆排。月柱起排。起运 = (出生日到最近节气天数 ÷ 3)

### Step 5：综合分析
`日主旺衰→格局→喜用神→大运→流年→历史校准→建议`

5. **命理分析仅供参考** — 人生在于自身的努力和选择
