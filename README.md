# 倪海厦全体系 · nihaixia-perspective

> 倪海厦「人纪 + 天纪」五术合一 Skill。中医经方 + 紫微斗数 + 易经64卦 + 阳宅风水 + 面相 + 八字。

## 安装

```bash
git clone https://github.com/T1ziyouren/nihaixia-perspective.git .claude/skills/nihaixia-perspective
```

## 包含内容

### 一、中医（人纪）

| 模块 | 内容 |
|------|------|
| 五大心智模型 | 物理医学、六大健康标准、阴实/阳不入阴、经方原方信仰、同病异治 |
| 感冒八经方 | 桂枝汤 / 麻黄汤 / 葛根汤 / 小青龙 / 大青龙 / 小柴胡 / 麻杏甘石 / 五苓散 |
| 癌症治疗三步 | 温阳 → 断营养 → 攻坚 |
| 诊疗决策 | 先分阴阳 / 重症先强心 / 治肾先强心 / 肝病通大肠 / 十三不治 |

### 二、紫微斗数（天纪·天机道）

基于 [iztro](https://github.com/SylarLong/iztro) 开源排盘引擎：

- **排盘九步**：四柱 → 命身宫 → 十二宫 → 天干 → 五行局 → 紫微星 → 十四主星 → 辅星 → 四化
- **批命要诀**：命好不如限好 / 吉处藏凶必凶 / 三方四正分析
- **十四主星**：北斗紫微天机太阳武曲天同廉贞 + 南斗天府太阴贪狼巨门天相天梁七杀 + 破军
- **格局**：紫府坐桓 / 七杀朝斗 / 巨日格 / 日月反背 / 半空折翅 等

### 三、易经64卦（人间道）

- 八卦体系、爻位判断、金钱卦占卜、六壬速断
- 卦图象解：鹿=禄位 / 虎=权威 / 马=调动 / 猴=阻碍 / 羊=回阳

### 四、阳宅风水（地脉道）

- 八卦方位与家庭成员对应
- 56种居住格局吉凶判断
- 凶宅格局（厕所在中间/西北厨房/西南厨房等）
- 改运方法：睡延年方、世代交替、最大宫原则

### 五、面相学

- 十二宫面部定位（命宫印堂→兄弟眉→夫妻眼尾→财帛鼻等）
- 气色判断：黄润喜事 / 黑暗劫杀 / 枯青死色
- 观人术：手势识谎、晏婴三才、气量鉴定

### 六、八字排盘

- 标准化8步信息收集流程
- 四柱排算（年立春/月节气/日万年历/时五鼠遁）
- 十神/藏干/十二长生完整体系
- 大运起运排列、历史事件校准
