## 概述

这是一个面向 AI 助手的专业技能（Skill），将倪海厦毕生所授「山、医、命、相、卜」五术整合为一个可调用的知识体系。Skill 定义角色扮演风格、排盘工作流、中医经方知识、易经占卜、阳宅风水、面相学六大模块，配合三个独立的 Python 排盘工具，实现从命盘数据到人生说明书的完整输出。

倪海厦（1954-2012），美籍华人中医师，以《人纪》系列中医课程和《天纪》系列命理课程闻名。本 Skill 是对其教学体系的数字化重建。

## 算法详解

### 一、八字排盘算法 (`tools/bazi_v1.4.py`)

#### 1.1 年柱
- **分界**：立春（通常2月4日前后）。立春前出生属上一年
- **年干**：`(公历年份 - 4) % 10`，映射到十天干
- **年支**：`(公历年份 - 4) % 12`，映射到十二地支

#### 1.2 月柱
- **月支**：以12节气为精确分界。寅=立春、卯=惊蛰、辰=清明、巳=立夏、午=芒种、未=小暑、申=立秋、酉=白露、戌=寒露、亥=立冬、子=大雪、丑=小寒
- **月干**：五虎遁法。以年干为索引查 `TIGER` 表获取该年寅月天干，逐月顺推

#### 1.3 日柱
- 以2000年1月1日为原点计算天数差
- **日干**：`(4 + 天数差) % 10`
- **日支**：`(6 + 天数差) % 12`
- 晚子时（23:00后）使用次日干支

#### 1.4 时柱
- **时支**：`时辰号 % 12`（子=0, 丑=1, ..., 亥=11）
- **时干**：五鼠遁法。以日干为索引查 `RAT` 表获取子时天干，逐时顺推

#### 1.5 十神
以日干为"我"，与其他天干比较五行生克：

| 关系 | 同五行 | 我生 | 我克 | 克我 | 生我 |
|------|--------|------|------|------|------|
| 同阴阳 | 比肩 | 食神 | 偏财 | 七杀 | 偏印 |
| 异阴阳 | 劫财 | 伤官 | 正财 | 正官 | 正印 |

地支十神取藏干主气（每地支第一个藏干）判定。

#### 1.6 藏干
十二地支内藏天干（古典口诀）：

| 地支 | 藏干 |
|------|------|
| 子 | 癸 |
| 丑 | 己癸辛 |
| 寅 | 甲丙戊 |
| 卯 | 乙 |
| 辰 | 戊乙癸 |
| 巳 | 丙庚戊 |
| 午 | 丁己 |
| 未 | 己丁乙 |
| 申 | 庚壬戊 |
| 酉 | 辛 |
| 戌 | 戊辛丁 |
| 亥 | 壬甲 |

#### 1.7 五行统计
- 天干权重 1.0
- 地支每个藏干权重 0.5（一支配1-3个藏干，故地支总权重约1.5）

#### 1.8 身强弱判定算法
综合评分公式：
```
总分 = 日主五行数×2
     + 月支得令(+2)          // 月支同日主五行
     + 藏干同日主数×1.5
     + 天干同类数×2
     - 克我五行数×1.5
```
分五档：
- 总分 ≥ 10 → 强
- 总分 ≥ 7.5 → 中强
- 总分 ≥ 5 → 中和
- 总分 ≥ 3 → 中弱
- 总分 < 3 → 弱

#### 1.9 喜用神
- **强/中强**：喜官杀（克我）、食伤（我生）、财（我克）；忌日主、印（生我）
- **弱/中弱**：喜印、日主；忌官杀、食伤、财
- **中和**：依据五行数量自动判定（多则忌、少则喜）

#### 1.10 大运
阳年男/阴年女 → 顺排；阴年男/阳年女 → 逆排。
起运年龄 = 距下/上一节气的天数 ÷ 3。从月柱开始，每步大运顺/逆推一柱，共8步，每步10年。

#### 1.11 地支关系
全面覆盖六合、三合、半合、六冲、六害、三刑。

#### 1.12 流年流月
- 流年干支 + 十神 + 与原局四柱的冲合害刑 + 与大运交互
- 流月：当年12个月按五虎遁排布，标注与原局的冲合伏吟

### 二、紫微斗数排盘算法 (`tools/ziweipan_v2.7.py`)

#### 2.1 命宫与身宫
- **命宫**：寅起正月 → 顺数到生月 → 逆数到生时
- **身宫**：寅起正月 → 顺数到生月 → 顺数到生时

#### 2.2 十二宫顺序
```
命宫 → 父母宫 → 福德宫 → 田宅宫 → 官禄宫 → 交友宫
→ 迁移宫 → 疾厄宫 → 财帛宫 → 子女宫 → 夫妻宫 → 兄弟宫
```

#### 2.3 五行局（纳音）
1. 由命宫天干地支计算：天干数 + 地支数
2. 超过5则减5，循环至≤5
3. 映射：1→木三局、2→金四局、3→水二局、4→火六局、5→土五局

#### 2.4 紫微星定位
核心算法：
1. 求最小偏移量 offset 使 `(农历日 + offset) % 局数 == 0`
2. `商 = (农历日 + offset) / 局数`
3. `紫微位置 = 商 - 1`
4. offset 为奇数则逆调整，为偶数则顺调整

#### 2.5 天府星定位
```
天府位置 = 12 - 紫微位置
```

#### 2.6 十四主星排布
- **紫微系**（逆排）：紫微→天机→(空)→太阳→武曲→天同→(空)(空)→廉贞
- **天府系**（顺排）：天府→太阴→贪狼→巨门→天相→天梁→七杀→(空)(空)(空)→破军

#### 2.7 四化
按年干查表获取禄/权/科/忌对应的主星：
| 年干 | 禄 | 权 | 科 | 忌 |
|------|-----|-----|-----|-----|
| 甲 | 廉贞 | 破军 | 武曲 | 太阳 |
| 乙 | 天机 | 天梁 | 紫微 | 太阴 |
| 丙 | 天同 | 天机 | 文昌 | 廉贞 |
| 丁 | 太阴 | 天同 | 天机 | 巨门 |
| 戊 | 贪狼 | 太阴 | 右弼 | 天机 |
| 己 | 武曲 | 贪狼 | 天梁 | 文曲 |
| 庚 | 太阳 | 武曲 | 太阴 | 天同 |
| 辛 | 巨门 | 太阳 | 文曲 | 文昌 |
| 壬 | 天梁 | 紫微 | 左辅 | 武曲 |
| 癸 | 破军 | 巨门 | 太阴 | 贪狼 |

#### 2.8 辅星系统（26颗）
| 辅星 | 定位方法 |
|------|----------|
| 文昌 | 戌起±时辰 |
| 文曲 | 辰起±时辰 |
| 地劫/地空 | 亥起±时辰 |
| 火星/铃星 | 年支分组+时辰偏移 |
| 禄存/擎羊/陀罗 | 年干定禄存，前后各一 |
| 天魁/天钺 | 年干定位 |
| 天马 | 年支定位 |
| 红鸾/天喜 | 年支定位，天喜=红鸾+6 |
| 左辅/右弼 | 农历月份定位 |
| 天刑/天姚 | 农历月份定位 |
| 三台/八座 | 左辅+日、右弼-日 |
| 恩光/天贵 | 文昌+日、文曲+日 |
| 龙池/凤阁 | 年支定位 |
| 天哭/天虚 | 年支定位 |

#### 2.9 长生十二神
年干定起点 → 阳男阴女顺排，阴男阳女逆排。

十二神：长生→沐浴→冠带→临官→帝旺→衰→病→死→墓→绝→胎→养

#### 2.10 博士十二神
从禄存位起 → 阳男阴女顺排，阴男阳女逆排。

十二神：博士→力士→青龙→小耗→将军→奏书→飞廉→喜神→病符→大耗→伏兵→官府

#### 2.11 大限
从命宫起，顺/逆排12宫，每宫覆盖10年（五行局数×10起算）。

### 三、农历转换 (`tools/lunar.py`)

基于 `lunardate` Python 库（0.2.x API），将公历日期转换为农历日期。

- 输入：标准 ISO 日期格式 `YYYY-MM-DD`
- 输出：农历年月日（含中文和数字格式）、是否闰月
- 范围：1900-2100

## 安装与依赖

```bash
pip install lunardate
```

Python 3.6+，纯标准库（除 `lunardate` 外）。

## 工具使用

```bash
# 第一步：公历转农历
python3 tools/lunar.py 1997-03-22

# 第二步：八字排盘（-a 全量分析模式）
python3 tools/bazi_v1.4.py -a 1997-03-22 8 女

# 第三步：紫微排盘（-l 农历参数模式）
python3 tools/ziweipan_v2.7.py -l 3 21 8 丁 丑 女

# 紫微排盘+星曜统计
python3 tools/ziweipan_v2.7.py -a 3 21 8 丁 丑 女

# 地支关系分析
python3 tools/ziweipan_v2.7.py -q rels 丑,卯,亥,申

# 藏干速查
python3 tools/ziweipan_v2.7.py -q 藏干 申
```

**时辰号对照**：子=0 丑=1 寅=2 卯=3 辰=4 巳=5 午=6 未=7 申=8 酉=9 戌=10 亥=11 晚子=12
